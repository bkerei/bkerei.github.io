In the JavaScript language, describe all the use cases for each of:

parentheses () - e.g. '1. Execute a function var result = doSomething()'
- () is were things get done. You state/give a thing a name like “var IceCream”, think of the () as where you tell it what you want to do with the ice cream for example var iceCream = (eat with a teaspoon).
brackets []
- [] is how I keep things in a nice tidy place like the multiple ways you can eat ice cream. So a code for that would be var = iceCream = (eat with a teaspoon [in bowl, cone, on a cat]).
braces {}
- {} define the start and end of the function, so:
var ice Cream = (eat with a teaspoon [in a bowl, cone, on a cat]) {
After dinner;
For breakfast;
Midnight snack;
}
single quotes ''
- Its states a string or these things are important in this list and are individual parts not one long things, so:
var ice Cream = (eat with a teaspoon [‘in a bowl’, ‘cone’, ‘on a cat’]) {
After dinner;
For breakfast;
Midnight snack;
}
double quotes ""
- Same as above just " " instead of ' '.
