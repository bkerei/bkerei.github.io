What is grid based design?

When I think about this all I hear is "This is Spinal Tap" in my head

I heard you like boxes, so we put boxes inside boxes!

If a site is not boxy enough you put more boxes in those boxes and you have a grid. With this grid system you can place your boxes within the boxes to make certain look.

Whats all the hype about responsive webpages?

The best way to answer this is to look at the two versions of Trade Me. This is the Desktop site and this is the Preview site.

The desktop site looks tiny on a moobile screen, while the Preview version moves and adapts to the persons screen. As more and more people use mobile devices to view the internet sites need to change to be viewable on a smaller screen. Responsiveness is how yoou do this.
