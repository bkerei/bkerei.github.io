9 July 2018

Which timebox and habits techniques did you try?

I tried the Timebox/Pomodoro mix up because I like the tomatoes and it looked interesting. I started with this blog and made sure I had a timer set to go off every 25 mins, then I'd set a timer for 5 mins where I picked up my crocheting for 5 mins. My objective was to get my blog sorted for the sprint and then create my CSS sheet. Finally get it all pushed to GitHub.It took a about 5 Pomodoros to achieve this.

What did you find interesting?

I got alot of crocheting done and my scarf is coming along nicely.

What worked for you?

My time management has never been an issue, if something needs to be done I make time for it. Right now I've made changes to my work and home enviroment to make sure I have time for this. I also have flow down naturally. I can focus on a task really well thanks to doing mindnumbing tasks at work for hours on end for years. I've managed to master the flow very well.

Which will you experiment further with?

I'll probaly stick with the 25 on 5 off method as its nice to break things up from time to time. Beyond that I use mindfulness in my daily life already and I realised I'd slipped a bit lately so I'm gonna work on getting back into that and loft thinking.
